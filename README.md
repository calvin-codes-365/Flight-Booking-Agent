Overview
This project is an interactive, console‑based flight booking assistant powered by a machine‑learning model. It predicts flight prices, generates possible flight options, recommends the best choices, and allows users to book flights—all through a simple menu-driven interface.

The system is built around a custom FlightAI class that uses a Random Forest Regressor to estimate flight prices based on distance, days before departure, and flight duration.

What I learned from building this project:

Basic machine learning workflows

Random forest regression

Building interactive console applications

Combining AI predictions with user-driven logic

Creating recommendation systems
